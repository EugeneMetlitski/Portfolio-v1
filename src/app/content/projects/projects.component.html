Portfolio Site

This is my portfolio site. I started making it in November 2018.

When examining this website, don't forget to resize the window width to see how it would work on tablet and mobile devices. The sidenav and top navbar work differently on scroll on different screen sizes.

The front-end is made using Angular 7 framework with scss, html and typescript code. The back-end is built using .Net Core 2.1 with c#.

The website is hosted on Azure and is served using .Net Core 2.1 api application and SQL Server database. The back-end api serves several front end Angular applications based on the route. The back-end also serves data to the front-end apps.

Click here to view the source code on GitHub for the front-end portion of Portfolio Site.

The back-end is backed up on GitBucket on a private repositary, because I don't want it to be on a public repository for security.
